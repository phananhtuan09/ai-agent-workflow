# Báo cáo đánh giá workflow

Evaluation artifacts produced by `docs/ai/project/WORKFLOW_EVALUATION_STANDARD.md`.

Mỗi file `docs/ai/workflow-evals/{name}.html` là một báo cáo HTML tiếng Việt, self-contained. Với workflow improvement, report ưu tiên session behavior, first divergence, rework, recovery, human intervention và cause attribution trước khi map vấn đề về workflow. Với adoption review không có trace, report dùng artifact-first evidence và controlled exercise.
