## Problem
Users need a way to archive completed notifications so the main list stays focused on active items.

## Solution
Allow users to archive a notification from the notifications list and keep archived notifications out of the default active view. Users should still be able to identify when an item has been archived and avoid archiving the same item multiple times.

## Acceptance Criteria
- [ ] AC1: A user can archive an active notification from the notifications list.
- [ ] AC2: A user does not see archived notifications in the default active notifications view.
- [ ] AC3: A user cannot archive the same notification multiple times from the same active view.

## Out of Scope
- Restoring archived notifications.
- Bulk archive actions.
